## Project Description 
Pet Connect is a social media application that is designed to connect pet owners. Users can create posts to plan dog walks with others, get potty training lessons, provide product information, and ask questions from other pet owners. If theyâ€™re lucky, they can even find soulmates for their pets.

### User Story
```
AS A person who owns a pet
I WANT a CMS-style blog/networking site
SO THAT I can connect with other pet owners and have discussions about my pet
```
